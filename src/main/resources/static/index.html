About Lawyd

Working off a nasty, long todo list is not rewarding in itself and at the end of the day there are still too many things left open. That is why we try to make it more pleasant, by not making the things simply disappear, but rather collecting them in a separate list. Besides the usual list of things to do, there is a list of things you have done to show you how awesome you are and how much you have rocked today.
